# Catálogo de Asensores

Proyecto React creado con [Vite](https://vitejs.dev/) y [pnpm](https://pnpm.io/).

## Uso rápido

Instala dependencias (asegúrate de tener [pnpm](https://pnpm.io/) instalado):

```bash
pnpm install
```

Luego inicia el servidor de desarrollo:

```bash
pnpm run dev
```

El sitio estará disponible normalmente en `http://localhost:5173`.

---

Estructura básica:

- `src/App.jsx`: Componente principal React.
- `src/main.jsx`: Entrada principal de la app.
- `public/index.html`: HTML principal.
- `vite.config.js`: Configuración de Vite.

¡Listo para comenzar a desarrollar!
